# API REST NODEJS (Express) y MYSQL

## Requirements

``` bash
Nodejs 18.13.0+
Express 4.18.2
MySQL 8.0.28
```

SERVER

``` bash
http://localhost:3000
```

API

``` BASH
GET:
/api/employee


GET:
/api/employee/4


POST:
/api/employee/

# Example

{
    "name": "Thor",
    "salary": 7100
}


PATCH:
/api/employee/4

# Example

{
    "salary": 5200
}


DELETE:
/api/employee/4
```
